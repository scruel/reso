# Reso-frontend
## 專案啟動方式

1. 安裝依賴：
   ```bash
   npm install
   ```
2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```
3. 開啟瀏覽器並前往 [http://localhost:3000](http://localhost:3000)
